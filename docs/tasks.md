
### afl-3.0

Generate a Academic Free License v3.0 `LICENSE` file in the current working directory.



**Example**

```sh
$ gen license:afl-3.0
$ gen license:afl-3.0 --dest ./docs
$ gen dest license:afl-3.0
```
